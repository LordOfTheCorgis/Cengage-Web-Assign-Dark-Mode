# Cengage Web Assign Dark Mode

Install the following browser extension:
https://chromewebstore.google.com/detail/stylus/clngdbkpkpeebahjckkjfobafhncgmne

Go into Web Assign and click the extension.

Press "Write New Style"

Add the code from the style.css.
